## 🚀 كيفية التشغيل

### 1. البنية التحتية
```bash
docker-compose up -d postgres redis
cd rtx4090_machine && python3 rtx_api_server.py
```

### 2. API Server
```bash
python3 -m uvicorn api.app:app --host 0.0.0.0 --port 8000
```

### 3. Desktop App
```bash
cd ui && npm run tauri dev
```
