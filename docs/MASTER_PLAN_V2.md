```
│    - الاستخدام: المعرفة المتراكمة       │
├─────────────────────────────────────────┤

│    - السعة: غير محدودة                  │
│    - المدة: للأبد                       │
│    - الاستخدام: كل شي (backup)          │
└─────────────────────────────────────────┘
```


```
    ↓

    ↓

    ↓
التدريب ← كل طبقة تتدرب
    ↓
النموذج اللغوي ← يتطور
    ↓

    ↓

    ↓
التنفيذ ← برمجة/بناء
    ↓
بعد الكارثة: تعليم + مراقبة + توجيه
```
